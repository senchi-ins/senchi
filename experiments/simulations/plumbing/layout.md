# Typical Canadian Single-Detached Home  
## Plumbing Layout & Pipe-Material Reference  
*“Rule-of-thumb spec for a mid-priced house built ≈ 1995 – present”*

---

## 1 . Snapshot of the House

| Attribute | Typical value | Notes |
|-----------|---------------|-------|
| Storeys | **2 above-grade + full/partial basement** | Two-storey plans dominate new construction. |
| Bedrooms | **3** | One often used as office/guest. |
| Bathrooms | **2 ½** (powder room + 2 full) | Always at least one WC per floor. |
| Toilets (WC) | **3** | Low-flow / dual-flush now code-minimum. |
| Laundry pairs | **1** | Basement in older homes; 2ᴺᴰ-floor in many post-2010 builds. |
| Dishwasher branch | **1** | Rough-in or installed. |

---

## 2 . Floor-by-Floor Plumbing Layout

| Floor / zone | Typical fixtures served | Common pipe sizes (nominal) | Why this layout? |
|--------------|------------------------|-----------------------------|------------------|
| **Roof penetration** | Vent stack | 2″ or 3″ ABS/PVC | Prevents trap siphoning. |
| **Upper floor** | • Ensuite: WC + lav + shower • Family bath: WC + lav + tub/shower | **Supply:** ½″ PEX/Cu **Drain:** 3″ (soil), 1½–2″ (branches) | Stacking baths over wet-wall cuts cost. |
| **Main floor** | • Powder room (WC + lav) • Kitchen sink + DW • Hose bibbs | **Supply:** ½″ branches off ¾″ trunk **Drain:** 2″ kitchen, 1½″ lav | Powder room near entry is resale must-have. |
| **Basement / lowest** | • Laundry standpipe • Water heater • Water meter / main shut-off • Floor drain | **Service line:** ¾–1″ Cu/PEX/HDPE **Building drain:** 3–4″ ABS/PVC | Equipment kept low for freeze protection and access. |
| **Site** | • 4″ sewer lateral to street/septic • Underground lawn service | **Sewer:** 4″ PVC SDR-35 <br>**Water:** ¾–1″ Type K Cu or CSA B137.1 HDPE | Below frost depth (≈ 1.2 m+). |

## 3 . Plumbing “Tree” with Materials Annotated

ROOF
│  2–3" ABS/PVC vent
└───┐
    │  UPPER FLOOR
    │  ├─ ensuite: WC/lav/shower  ← ½" PEX hot/cold
    │  └─ family bath: WC/lav/tub ← ½" PEX hot/cold
    │       ↳ 1½–2" ABS drains to 3" soil
    ├───┄ 3" ABS/PVC soil stack
    │  MAIN FLOOR
    │  ├─ powder WC/lav          ← ½" PEX
    │  └─ kitchen sink + DW      ← ½" PEX hot/cold
    │       ↳ 2" ABS drain + 2" vent
    ├───┄ continues to basement
    │  BASEMENT
    │  ├─ laundry stand-pipe     ← ½" PEX
    │  └─ floor drain
SERVICE ENTRY  ← ¾–1" Type K Cu / HDPE → Meter → Shutoff Valve → [SENSOR] → ¾" PEX trunk → ½" branches
BUILDING DRAIN → 4" ABS/PVC → 4" PVC SDR-35 sewer lateral → municipal sewer/septic
